# Docker Compose - Passo a Passo"""# Docker Desktop 1) Abra Docker Desktop2) Faça o Login: # VSCODE 1) Abra uma pasta 2) De o nome de DDocker3) Abra um arquivo e digite: docker-compose.yml4) Copie e cole o código abaixo: 5) Vá ao terminal e digite: cd DDocker6) Digite: ls7) Digite: docker-compose up # Docker Desktop8) Da um play(running) # Dbeaver 9) Abra o Dbeaver10) Abra um arquivo  MYSQL 11) MYSQL_DATABASE: lucas_teste12) MYSQL_USER: usuario13) MYSQL_ROOT_PASSWORD: luquinh@S00714) Conecte """ version: '3.9'services:  mysql_DDocker:    container_name: mysql_DDocker    hostname: mysql_DDocker    image: mysql:8    restart: always    command:      - --authentication-policy=mysql_native_password      - --character-set-server=utf8mb4      - --collation-server=utf8mb4_unicode_ci      - --innodb_force_recovery=0    volumes:      - ./mysql_DDocker:/var/lib/mysql    ports:      - 3306:3306    environment:      MYSQL_ROOT_PASSWORD: luquinh@S007      MYSQL_DATABASE: lucas_teste      MYSQL_USER: usuario      MYSQL_PASSWORD: luquinh@s007      TZ: America/Sao_Paulo
